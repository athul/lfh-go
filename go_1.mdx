- `assign and use or on-the-go use`

```go 6[6,7],7[6,7] title="Declare and Assign" subtitle="The ':=' operator is used for assigning variables without declaration."

	i:=10
	s:="Canada"

	fmt.Println(i)
	fmt.Println(s)
}
```

```go title="You can declare and Assign multiple variables like this" subtitle="https://play.golang.org/p/GG9Q9GFTSC_r"
package main

import "fmt"

func main() {
	var fname, lname string = "John", "Doe"
	m, n, o := 1, 2, 3
	item, price := "Mobile", 2000

	fmt.Println(fname + lname)
	fmt.Println(m + n + o)
	fmt.Println(item, "-", price)
}
```

## Strings in GO

---
### Strings can be formatted in go with
- \` string  \` (backticks)
- "  string  "  (Double Quotes)
- '  string  '  (Single Quotes)

---


<CodeSurfer>

```go title="Strings formatted with (backtick) always display the same way they are written or Raw string literal" 
package main

import "fmt"

func main() {
    fmt.Println(`Let's print out
	
this string.`)
}
```
</CodeSurfer>

---

<CodeSurfer>

```go title="Strings formatted with (double quotes) can be highlt formatted with string literals" 
package main

import "fmt"

func main() {
    fmt.Println("Let's print out this string.")
}
```
```go 6 title="With New line literal" subtitle="The output will have a newline break b/w 'out' and 'this'"
package main

import "fmt"

func main() {
    fmt.Println("Let's print out \n this string.")
}
```
```go 6 title="With Tab literals" subtitle="The output will have a tab space(6 spaces)  b/w 'out' and 'this'"

    fmt.Println("Let's print out \t this string.")
}
```
